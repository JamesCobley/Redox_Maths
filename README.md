# Redox_Maths
# Supporting Materials – A Mathematical Theory of Redox Biology

## Overview

This repository contains supporting materials for the manuscript:

**Cobley & Nikolaidis – A Mathematical Theory of Redox Biology**

The manuscript presents a formal theoretical framework for redox biology based on admissible state spaces, flux-weighted network structure, dynamical evolution on flux space, and spatial reaction–transport embedding.

The materials provided here allow reproduction of the formal constructions, worked examples, and matrix formulations used in the manuscript and supplemental notes.

This work is theoretical and does not rely on experimental datasets. The materials provided here support reproducibility of the formal framework .
